Vireoka Teaser Web – Phase 1

To preview the site locally:

  cd vireoka_web
  python3 -m http.server 8000

Then open:

  http://localhost:8000/

Key entry points:
  - /                → Main umbrella teaser (Vireoka Home)
  - /subdomains/dating/      → Niche Dating Platform Creator teaser
  - /subdomains/stablecoin/  → Quantum-Secure Stablecoin teaser
  - /blog/                   → Phase 2 blog shell (Founder's Journey)
  - /whitepapers/            → Phase 2 whitepaper gate shell
